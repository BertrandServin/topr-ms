topr : an R package for viewing and annotating genetic association data

See full documentation at https://wuxi-nextcode.github.io/topr/ ### Installation

Install from CRAN:

install.packages("topr")

Or from github:

devtools::install_github("wuxi-nextcode/topr")

And then load the package:

library(topr)
library(tidyverse)
library(ggrepel)

Example usage

In this example we demonstrate the basic usage of the topr library.

Example input datasets

The topr package comes with three example datasets; CD_UKBB and UC_UKBB which are subsets of association results (SNPs are filtered on P<1e-03) for Crohn´s disease and Ulcerative colitis from the UK biobank, and CD_FINNGEN which is a subset of association results (P<1e-03) of Crohn’s from FinnGen.

The input dataset must include least three columns (CHROM, POS and P), where naming of the columns is flexible (i.e the chr label can be either chr or chrom and is case insensitive).

The chromosome in the CHROM column can be represented with our without the chr suffix, e.g (chr1 or 1)

head(CD_UKBB) %>% select(CHROM,POS,P)

Input data column names and alternatives understood by topr:

Required columns and alternative namings:
CHROM:          CHR,chr,chrom,chromosome,CHROMOSOME
POS:            pos,BP,bp,base_pair_location
P:              p,PVAL,pval,PVALUE,pvalue,p_value

Optional columns and alternatives:
ID:             rsid,rsId,RSID,SNP,snp,rsName,rsname,RSNAME
Gene_Symbol:    Gene_symbol,GENENAME,geneName,genename,GENE,gene
Max_Impact:     max_impact,Impact,impact
REF:            ref
ALT:            alt
BETA:           beta,b,B
OR:             or,odds_ratio

It is highly recommended to check the number of datapoints in your dataset before you plot, since a very large dataset will take a long time to plot.

paste("Number of SNPs in the dataset: [", length(CD_UKBB$POS),"]", sep = "")

If your dataset is very large, you reduce it by filtering variants by their p-values

gwas_smaller <- CD_UKBB %>% filter(P < 1e-05)

Manhattan plots

Get an overview of association results for crohn’s disease (CD) on a Manhattan plot

manhattan(CD_UKBB)

Label the top SNPs with the name of their nearest gene

Use the annotate argument in the manhattan function to label the top SNPs with p-values below the annotate threshold with their nearest gene

manhattan(CD_UKBB, annotate = 5e-09)

Change the color, size and angle of the labels. Move the labels along the x and y - axes with the ggrepel parameters (nudge_x and nudge_y)

manhattan(CD_UKBB, annotate = 5e-09, angle=25, label_color="gray40",label_size=3.5, nudge_x = 100000000, nudge_y=2)

Highlight genes of interest at the base of the plot

manhattan(CD_UKBB, annotate = 5e-09, highlight_genes = c("NOD2","IL23R","JAK2"), angle=25)

The significance threshold

Remove the significance threshold

manhattan(CD_UKBB, annotate = 5e-09, sign_thresh = NULL)

Use multiple significance thresholds and set different colors for each

manhattan(CD_UKBB, annotate = 5e-09, sign_thresh = c(1e-07,1e-09,1e-11), sign_thresh_color = c("red","orange","darkorange"))

View one chromsome only

Take a closer look at the results by chromosome. Here we plot the results on chromosome 1 only.

manhattan(CD_UKBB, annotate = 1e-09, chr = "1")

Regionplot

Zoom in further on the chromosome plot with the regionplot function.

Zoom in on a gene of interest, e.g IL23R:

regionplot(CD_UKBB, gene="IL23R", annotate= 1e-09)

Zoom in on the top hit on a chromosome

CHR <- "chr16"
top_hit <- get_top_hit(CD_UKBB,chr=CHR)
regionplot(CD_UKBB, chr = CHR, xmin=top_hit$POS-150000 ,xmax= top_hit$POS+150000)

Display multiple GWAS results on the same plot

One or more GWAS results can be plotted together on the same plot

Manhattan multiple phenotypes/GWASes

manhattan(list(UC_UKBB,CD_UKBB,CD_FINNGEN),legend_labels = c("UC UKBB","CD UKBB","CD Finngen"),title="IBD")

Annotate top variants

manhattan(list(UC_UKBB,CD_UKBB,CD_FINNGEN),annotate=1e-12,legend_labels = c("UC UKBB","CD UKBB","CD Finngen"),title="IBD")

Set different annotation thresholds for each study

manhattan(list(UC_UKBB,CD_UKBB,CD_FINNGEN),annotate=c(1e-15,1e-12,5e-9),legend_labels = c("UC UKBB","CD UKBB","CD Finngen"),title="IBD")

ntop

Use the ntop argument to control how many GWASes appear on the top panel of the plot

manhattan(list(CD_UKBB,CD_FINNGEN,UC_UKBB),ntop=2,annotate=c(1e-10,1e-9,1e-12),legend_labels = c("CD UKBB","CD Finngen","UC UKBB"),title="IBD", sign_thresh=c(5e-07,5e-09), sign_thresh_color = c("red","orange"))

Regionplot multiple phenotypes/GWASes

regionplot(list(CD_UKBB,CD_FINNGEN),gene="NOD2", annotate=c(1e-15, 1e-09), legend_labels = c("UKBB", "Finngen"), title="Crohn's disease (CD)")
manhattan(list(CD_UKBB,CD_FINNGEN,UC_UKBB),legend_labels = c("CD UKBB","CD Finngen","UC UKBB"),title="IBD")

The ntop argument

Use the ntop argument to set the number of datasets displayed at the top (default value is 3)

manhattan(list(CD_UKBB,CD_FINNGEN,UC_UKBB),ntop=2,legend_labels = c("CD UKBB","CD Finngen","UC UKBB"),title="IBD")

Useful functions

get_top_hit(CD_UKBB, chr="chr16", eval=FALSE)
dat1 <- get_best_snp_per_MB(CD_UKBB,thresh = 1e-07, region=1000000)
#get overlapping SNPS overlapping in two datasets
overlapping_snps <- dat1 %>% get_overlapping_snps_by_pos(CD_FINNGEN)

overlapping_snps_matched <- overlapping_snps %>% match_alleles()
overl_snps_matched_pos_allele_dat1 <- overlapping_snps_matched %>% flip_to_positive_allele_for_dat1()
snpset1 <- overl_snps_matched_pos_allele_dat1 %>% annotate_with_nearest_gene(protein_coding_only = T)

#or do all this in one go, by calling the create_snpset functon
snpset1 <- create_snpset(CD_FINNGEN, CD_UKBB, thresh = 1e-06)
snpset2 <- create_snpset(CD_UKBB, CD_FINNGEN, thresh= 1e-06)
e1 <- effect_plot(snpset1, pheno_x="CD Finngen", pheno_y="CD UKBB",color=get_topr_colors()[1], gene_label_thresh = 1)
e2 <- effect_plot(snpset2, pheno_x="CD UKBB", pheno_y="CD Finngen", color=get_topr_colors()[2], gene_label_thresh=1,annotate_with = "ID")
grid.arrange(e1,e2)
